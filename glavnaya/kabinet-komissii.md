### **Разделы кабинета:** 

-  Мои голосования

-  Договоры

-  Голосование по предварительному допуску

-  Голосование по допуску

-  Голосование по протоколу итогов



### Мои голосования

<https://dev.eurasiantech-tender.kz/Account/Commission/MyPurchases>

Отображается таблица объявлений, в которых пользователь участвовал в роли одного из членов комиссии.

При переходе к элементу таблицы (нажать на номер строки) открывается страница просмотра объявления

<https://dev.eurasiantech-tender.kz/Account/Purchase/PurchaseDetails/1420>









### Договоры

<https://dev.eurasiantech-tender.kz/Account/Commission/Contracts?status=Created>

Отображается таблица со списком страниц договоров, в объявлениях которых пользователь участвовал как член комиссии. 



При переходе по номеру строки открывается страница с ссылкой <https://dev.eurasiantech-tender.kz/Account/Commission/Contract/1400> 











### Голосования комиссии 



На страницах отображаются список страниц для голосования чл. комиссии по разным протоколам: допуск, пред.допуск и итогов. 



-  Голосование по предварительному допуску <https://dev.eurasiantech-tender.kz/Account/Commission/CommissionList?commissionType=PreAdmission>

При нажатие на номер строки открывается страница 

<https://dev.eurasiantech-tender.kz/Account/Purchase/PreAdmissionCommissionVoting/713>



-  Голосование по допуску <https://dev.eurasiantech-tender.kz/Account/Commission/CommissionList?commissionType=AdmissionProtocol>

При нажатие на номер строки открывается страница <https://dev.eurasiantech-tender.kz/Account/Purchase/AdmissionProtocol/1312>



-  Голосование по протоколу итогов <https://dev.eurasiantech-tender.kz/Account/Commission/CommissionList?commissionType=Summarizing>

При нажатие на номер строки открывается страница <https://dev.eurasiantech-tender.kz/Account/Purchase/Summarizing/773>
