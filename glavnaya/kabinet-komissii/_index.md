title: Кабинет комиссии
---

Кабинет комиссии предназначен для работы членов комиссии с закупками: просмотра объявлений, участия в голосованиях и работы с договорами.

---

## Разделы кабинета

В кабинете доступны следующие разделы:

-  Мои голосования

-  Договоры

-  Голосование по предварительному допуску

-  Голосование по допуску

-  Голосование по протоколу итогов

---

## 1\. Мои голосования

Раздел содержит список закупок, в которых пользователь участвует как член комиссии.

### Что отображается

Таблица с объявлениями:

-  номер закупки

-  наименование

-  статус

### Как открыть закупку

1. Нажмите на номер строки

2. Откроется страница просмотра объявления

---

## 2\. Договоры

Раздел содержит список договоров по закупкам, где пользователь входит в комиссию.

### Что отображается

Таблица договоров:

-  номер

-  статус

-  связанная закупка

### Как открыть договор

1. Нажмите на номер строки

2. Откроется страница договора

---

## 3\. Голосование комиссии

Разделы голосования используются для принятия решений по закупке на разных этапах.

---

### 3\.1 Голосование по предварительному допуску

Используется для первичной оценки заявок.

### Действия

1. Перейдите в раздел

2. Выберите закупку

3. Нажмите на номер строки

4. Откроется страница голосования

---

### 3\.2 Голосование по допуску

Используется для окончательного решения о допуске участников.

### Действия

1. Перейдите в раздел

2. Выберите закупку

3. Нажмите на номер строки

4. Откроется страница голосования

---

### 3\.3 Голосование по протоколу итогов

Используется для утверждения итогов закупки.

### Действия

1. Перейдите в раздел

2. Выберите закупку

3. Нажмите на номер строки

4. Откроется страница голосования

---

## Общая логика работы

Во всех разделах действует одинаковый принцип:

-  отображается список закупок

-  переход осуществляется по номеру строки

-  открывается детальная страница

---

## Важно

-  Голосование доступно только членам комиссии

-  Решения фиксируются в системе

-  После завершения этапа голосование может быть недоступно

---

## Рекомендации

-  Проверяйте документы перед голосованием

-  Убедитесь, что вы на правильном этапе процедуры

-  Следите за сроками голосования

---

## Результат

После завершения голосований:

-  формируются протоколы

-  принимаются решения по закупке

-  процедура переходит к следующему этапу
