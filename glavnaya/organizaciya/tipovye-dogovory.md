# 

Раздел **«Типовые договоры**» позволяет создавать договоры автоматически на основе готовых шаблонов.

Это упрощает и ускоряет процесс подготовки документов после завершения закупки.

---

## Когда используется

После завершения закупки заказчик может:

-  создать договор вручную

-  загрузить файл

-  или использовать шаблон

---

## Как создать договор из шаблона

### Шаг 1. Перейдите в договоры

Откройте страницу **Договоры** по закупке

---

### Шаг 2. Нажмите «Добавить из шаблона»

Откроется окно выбора шаблона

---

### Шаг 3. Выберите шаблон

Доступны варианты:

-  Договор поставки

-  Типовой договор

-  Типовой договор по услугам

---

### Шаг 4. Заполните данные

После выбора откроется форма с:

-  предпросмотром документа

-  полями для заполнения

---

## Общая логика заполнения

### Автоматически заполняется

-  данные заказчика

-  данные поставщика

-  сумма договора

-  условия закупки

-  реквизиты

---

### Заполняется вручную

-  номер договора

-  дата

-  дополнительные условия

-  реквизиты (при необходимости)

---

## Важное правило

Сумма договора должна быть:

-  больше 0

Если указать 0 -- система не позволит сохранить документ.

---

## Типовой договор

Используется для закупки товаров.

### Основные поля:

**О документе:**

-  название

-  номер

-  дата

-  сумма

-  валюта

**Данные сторон:**

-  заказчик (заполняется автоматически)

-  поставщик (заполняется автоматически)

**Приложение:**

-  условия оплаты

-  условия поставки

-  срок поставки

-  гарантия

---

## Типовой договор по услугам

Используется для закупки услуг.

### Дополнительно:

-  описание услуг

-  ссылка на протокол итогов

-  должности подписантов

---

## Особенности

-  часть полей недоступна для редактирования

-  такие поля выделены визуально

-  данные берутся из карточки организации и закупки

---

## Результат

После заполнения:

1. Нажмите **«Добавить»**

2. Договор создаётся

3. Появляется в списке договоров

4. Доступен для подписания

---

## Рекомендации

-  используйте шаблоны для ускорения работы

-  проверяйте автоматически подставленные данные

-  корректируйте поля перед сохранением

---

## Важно

-  шаблон не заменяет проверку данных

-  ответственность за корректность данных остаётся у пользователя

-  перед подписанием обязательно проверьте документ
